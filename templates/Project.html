Google Colab

เราสามารถใช้ Google Colab สร้าง Notebook เขียนโปรแกรมภาษา Python ได้ฟรี ๆ และแถมยังมี GPU, TPU ให้เราได้ใช้ฟรีอีก ทีละ 12 ชั่วโมง เป็นตัวช่วยในการทำงานเรื่องของการ Train Model ของตัว Yolo

Visual Studio Code

เป็นโปรแกรมประเภท Editor ใช้ในการแก้ไขโค้ดที่มีขนาดเล็ก แต่มีประสิทธิภาพสูง เป็น OpenSource โปรแกรมจึงสามารถนำมาใช้งานได้โดยไม่มีค่าใช้จ่าย เหมาะสำหรับนักพัฒนาโปรแกรมที่ต้องการใช้งานหลายแพลตฟอร์ม ซึ่งในที่นี้เราใช้ในการทำ Html Css และ Javascript ควบคู่กับการทำ Flask เพื่อนำตัว Model ขึ้นเว็บไขต์

Flask

web framework ที่เขียนขึ้นมาสำหรับ Python เพื่อใช้ร่วมกัน webserver ในที่นี้คือเราทำมาใช้เป็นตัวในการนำ Model Machine Learning ขึ้นเว็บไขต์ และเป็นสื่อกลางการทำงานของ 2 ส่วนสำคัญ คือ เว็บไชต์ และ Machine Learning
